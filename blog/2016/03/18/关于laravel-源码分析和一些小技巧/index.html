关于Laravel 源码分析和一些小技巧

2016年03月18日 in coding using tags Laravel

Laravel 的多语言是可以覆盖的，app.php里面定义的fallback_locale是基础语言包，如果一个词在当前语言包里面没有的时候就会去基础语言包里面找

Laravel 的 Cookie是经过加密的，任何本地改动都会造成cookie失效 赋值方法是

Cookie:queue(name,value,min);

取值方法

$request->cookie[name];

如果想不加密，就还是用PHP原生方法吧

Kernel中定义的web中间件有四个

EncryptCookies 加密和解谜Cookie

AddQueuedCookiesToResponse 写入Cookie 就是之前queueinquiry的

StartSession 相当于session_start

VerifyCsrfToken 方式CSRF攻击的

api的就只有ThrottleRequests一个，主要是用来限制单位时间内请求次数的防止被刷
